# 🌱 영감노트 (Inspiration Note) - 보안 에디션

영감노트는 스쳐 지나가는 아이디어와 영감을 안전하게 기록하고 관리할 수
있는 웹 애플리케이션입니다.\
Google Spreadsheets를 데이터베이스로 활용하여 서버 비용 없이 데이터를
영구 보존할 수 있으며, 네트워크가 없는 환경에서도 오프라인 모드를 통해
안전하게 사용할 수 있습니다.

------------------------------------------------------------------------

## ✨ 주요 기능

### 하이브리드 저장소

-   온라인 모드: Google Apps Script(GAS)를 통해 개인 Google
    스프레드시트에 데이터를 저장합니다.\
-   오프라인 모드: 서버 연동 없이 브라우저 로컬 스토리지에만 데이터를
    저장할 수 있습니다.

### 다양한 보기 모드

-   직관적인 카드형(Grid) 뷰\
-   가독성 높은 리스트(List) 뷰

### 강력한 첨부파일 관리

-   이미지 및 문서 파일 업로드 지원\
-   Google Drive 연동 시 다운로드 없이 미리보기(Preview) 제공

### 데이터 백업 / 복원

-   JSON 파일로 데이터 내보내기 / 불러오기 가능\
-   데이터 유실 방지

### 보안 강화

-   사용자가 직접 설정한 비밀번호(Access Key)를 통해서만 데이터 접근
    가능

------------------------------------------------------------------------

## 🛠️ 설치 및 설정 방법 (How to Setup)

이 앱은 프론트엔드(index.html)와\
백엔드(Google Apps Script)로 구성되어 있습니다.

------------------------------------------------------------------------

## 1️⃣ 백엔드 설정 (Google Apps Script)

1.  새 Google Spreadsheet를 생성합니다.\
2.  상단 메뉴에서 확장 프로그램 → Apps Script를 클릭합니다.\
3.  제공된 Code.gs 내용을 붙여넣습니다.\
4.  중요: 스크립트 상단의 ACCESS_KEY 값을 본인만의 비밀번호로
    변경하세요.

``` javascript
const ACCESS_KEY = "여기에_비밀번호_입력"; // 예: "mySecret1234"
```

5.  우측 상단 배포 → 새 배포를 클릭합니다.
    -   유형: 웹 앱\
    -   다음 사용자 권한으로 실행: 나(Me)\
    -   액세스 권한: 모든 사용자(Anyone) (중요)
6.  배포 후 생성된 웹 앱 URL을 복사해 둡니다.

------------------------------------------------------------------------

## 2️⃣ 프론트엔드 설정 (Web App)

1.  index.html 파일을 열거나 호스팅 서버에 업로드합니다.\
2.  앱 우측 상단의 설정(⚙️) 아이콘을 클릭합니다.\
3.  서버 연동 탭에서 복사한 웹 앱 URL과 설정한 액세스 키를 입력하고
    연동합니다.\
4.  서버 없이 사용하려면 오프라인으로 시작을 선택하세요.

------------------------------------------------------------------------

## ⚠️ 보안 유의사항

### Access Key 관리

-   Code.gs에 설정한 비밀번호와 index.html에 입력하는 비밀번호는
    타인에게 노출되지 않도록 주의하세요.

### 코드 공유 시

-   GitHub 등에 코드를 업로드할 경우 ACCESS_KEY와 Web App URL을
    비워두거나 가명 처리하여 업로드하시기 바랍니다.
